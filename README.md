MY STUDY TRACKER

My Study Tracker is a console application in Python that can help students to follow their learning time up.
The user can add subjects of his choice, he can create study sessionspeut ajouter des matières, enregistrer des sessions d’étude et voir le temps total passé à étudi
 
Technologies utilisées
Ce projet utilise :
•	Python
•	Programmation Orientée Objet (OOP)
•	JSON pour stocker les données
•	API REST (ZenQuotes)
•	Bibliothèque requests
